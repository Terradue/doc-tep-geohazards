# Overview

A *Thematic App* is a specific version of the geobrowser, in which specific parameters are pre-defined, in order to serve a specific thematic.
It is defined by a list of services offerings, such as:

> - web widget,
> - map layer,
> - map background,
> - data series,
> - wps services
